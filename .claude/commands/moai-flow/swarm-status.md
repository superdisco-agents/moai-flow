# /swarm-status

Display current moai-flow swarm status and performance metrics.

## Usage

```
/swarm-status
```

## Parameters

None

## Examples

```
/swarm-status
```

## Behavior

1. Query current swarm session
2. Retrieve topology and agent states
3. Collect performance metrics
4. Format status report

## Output

Returns comprehensive status including:
- Swarm session ID
- Current topology type
- Active/inactive agents
- Consensus state
- Performance metrics (throughput, latency)
- Recent errors or warnings
