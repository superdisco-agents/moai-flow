## 🔧 기술 세부사항

### D2Coding 폰트 설치 수정

**이전 (작동 안 함)**:
```bash
brew tap homebrew/cask-fonts  # ❌ Deprecated (2024년 5월)
brew install --cask font-d2coding
```

**현재 (작동함)**:
```bash
brew install --cask font-d2coding-nerd-font  # ✅ No tap required
```

**변경 이유**:
- Homebrew cask-fonts tap이 2024년 5월 deprecated됨
- 모든 폰트가 메인 `homebrew/cask` 저장소로 이동
- D2Coding Nerd Font는 프로그래밍 아이콘과 ligature 포함
- 3배 더 인기 있음 (3,969 vs 1,297 annual installs)

### Ghostty 설정 최적화

**폰트 폴백 체인**:
```ini
font-family = "D2CodingLigature Nerd Font"  # 한글 전용
font-family = "JetBrains Mono"               # 영문 폴백
font-size = 14
font-feature = "calt"  # Contextual alternates
font-feature = "liga"  # Ligatures
```

**최적화 포인트**:
- D2Coding을 기본 폰트로 설정 (한글 우선)
- JetBrains Mono를 폴백으로 설정
- Ligature 활성화 (프로그래밍 심볼)
- 적절한 폰트 크기 (14pt)

## 🚀 사용 방법

### 빠른 시작 (3단계)

**1단계: Ghostty 한글 폰트 설정 (2분)**
```bash
cd _config/MOAI-ADK-KO
bash scripts/setup-korean-environment.sh
```

**2단계: MoAI-ADK 설치 (5-10분)**
```bash
cd /path/to/moai-adk/moai-adk
python3 -m venv .venv
source .venv/bin/activate
uv pip install -e .
```

**3단계: 설치 확인 (1분)**
```bash
moai-adk --version
moai-adk doctor
moai-adk status
```
