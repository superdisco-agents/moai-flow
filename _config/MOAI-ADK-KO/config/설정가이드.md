# MoAI-ADK 한글 전용 설정 가이드

이 디렉토리는 MoAI-ADK의 한글 전용 모드 설정을 포함합니다.

## 📋 파일 목록

### `config.json`
MoAI-ADK 한글 전용 구성 파일

**주요 설정:**
- `conversation_language: "ko"` - 대화 언어를 한국어로 강제
- `enforce_language_only: true` - 영어 폴백 비활성화
- `cli_output: "ko"` - CLI 출력을 한글로 설정
- `locale: "ko_KR"` - 한국 로케일
- `timezone: "Asia/Seoul"` - 서울 시간대

## 🎯 용도

이 설정 파일을 사용하면:
1. ✅ 모든 MoAI-ADK 출력이 한글로 표시됩니다
2. ✅ 영어 폴백이 비활성화됩니다
3. ✅ CLI 명령어와 오류 메시지가 한글로 표시됩니다
4. ✅ 한국 시간대와 로케일이 설정됩니다

## 📦 설치

이 설정을 프로젝트에 적용하려면:

```bash
# MoAI-ADK 설정 디렉토리로 복사
cp _config/MOAI-ADK-KO/config/config.json .moai/config/config.json

# 또는 심볼릭 링크 생성
ln -s $(pwd)/_config/MOAI-ADK-KO/config/config.json .moai/config/config.json
```

## ⚙️ 설정 적용 확인

```bash
# MoAI-ADK 상태 확인 (한글로 출력됨)
moai-adk status

# 구성 파일 확인
cat .moai/config/config.json
```

## 🔧 커스터마이징

필요에 따라 `config.json`을 수정할 수 있습니다:

```json
{
  "language": {
    "conversation_language": "ko",
    "enforce_language_only": true
  },
  "messages": {
    "cli_output": "ko",
    "error_messages": "ko"
  }
}
```
