# 🚀 MoAI-ADK 한글 버전 빠른 시작 가이드

## 개요
이 가이드는 MoAI-ADK를 한글 환경에서 빠르게 설치하고 사용하는 방법을 안내합니다.

---

## 1단계: Ghostty 한글 폰트 설정 (2분)

### 자동 설정 (권장)
```bash
bash _config/MOAI-ADK-KO/scripts/setup-korean-environment.sh
```

### 수동 설정
```bash
brew tap homebrew/cask-fonts
brew install --cask font-d2coding
```

Ghostty 설정 파일 (`~/.config/ghostty/config`) 편집:
```ini
font-family = "D2Coding"
font-size = 14
```

---

## 2단계: MoAI-ADK 설치 (5-10분)

상세한 설치 가이드를 참조하세요:
```bash
cat _config/MOAI-ADK-KO/INSTALL-MOAI-ADK.md
```

**간단 설치**:
```bash
cd /path/to/moai-adk/moai-adk
uv pip install -e .
```

---

## 3단계: 설치 확인 (1분)

### 가상 환경 활성화
```bash
source .venv/bin/activate
```

### 버전 확인
```bash
moai-adk --version
# 출력: MoAI-ADK, version 0.30.2
```

### 상태 점검
```bash
moai-adk doctor
# 모든 항목이 ✓ 표시되어야 합니다
```

---

## 주의사항 ⚠️

### 프로젝트 수준 설치
- ✅ MoAI-ADK는 **프로젝트 수준**으로 설치됩니다
- ✅ **가상 환경 활성화**가 필수입니다
- ❌ **전역 설치가 아닙니다**

### 사용 방법
```bash
# 항상 가상 환경을 먼저 활성화
source .venv/bin/activate

# 그 다음 moai-adk 명령어 사용
moai-adk status
moai-adk --help
```

---

## 문제 해결

### 한글이 깨져 보일 때
1. D2Coding 폰트 설치 확인: `fc-list | grep D2Coding`
2. Ghostty 설정 확인: `cat ~/.config/ghostty/config`
3. Ghostty 재시작: `Cmd+Shift+,`

### 명령어를 찾을 수 없을 때
1. 가상 환경 활성화 확인: `which python3`
2. 설치 확인: `pip list | grep moai-adk`
3. 경로 확인: `.venv/bin/moai-adk --version`

### 추가 도움말
- 전체 설치 가이드: `INSTALL-MOAI-ADK.md`
- 체크리스트: `INSTALLATION-CHECKLIST.md`
- 제거 가이드: `docs/UNINSTALL-CLAUDE-FLOW.md`
