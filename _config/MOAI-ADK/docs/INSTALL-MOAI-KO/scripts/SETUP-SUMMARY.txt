CUSTOMIZATION OPTIONS
================================================================================

CHANGE FONT:
  1. Open config: nano ~/.config/ghostty/config
  2. Edit line: font-family = [your font]
  3. Options: See INSTALLATION-GUIDE.md
  4. Restart Ghostty

CHANGE FONT SIZE:
  1. Open config: nano ~/.config/ghostty/config
  2. Edit line: font-size = [your size]
  3. Try: 12-14 for regular, 10-11 for Retina, 16-18 for accessibility
  4. Restart Ghostty

CHANGE COLORS:
  1. Open config: nano ~/.config/ghostty/config
  2. Modify:
     background = #1e1e2e
     foreground = #cdd6f4
     cursor-color = #89b4fa
  3. Restart Ghostty
