QUICK START (5 MINUTES)
================================================================================

The easiest way to get started:

  1. Open Terminal: Cmd+Space, type "Terminal", press Enter

  2. Go to scripts directory:
     cd /Users/rdmtv/Documents/claydev-local/agent-os-v2/moai-adk/_config/MOAI-ADK/docs/INSTALL-MOAI-KO/scripts

  3. Run complete setup:
     ./setup-all.sh

  4. Follow the prompts:
     • Type 'y' to start installation
     • Select font (choose 1 for Noto Sans Mono CJK KR)
     • Enter font size (press Enter for default 12)
     • Review config if you want
     • Let it restart Ghostty

  5. Done! Test Korean text:
     Open Ghostty and type: echo "안녕하세요"

That's it! You now have Korean font support.

================================================================================
COMMON QUESTIONS
================================================================================

Q: Is this safe?
A: Yes! Operations are user-only, configuration backups are automatic,
   and everything is reversible.

Q: What if something goes wrong?
A: All scripts have error handling and detailed logging. Check the log
   files for details, or run ./verify-setup.sh for diagnostics.

Q: Can I uninstall?
A: Yes. Remove fonts via Homebrew, restore config from backup.
   See INSTALLATION-GUIDE.md for details.

Q: Do I need Ghostty installed first?
A: Helpful but not required. The script checks and provides instructions
   if Ghostty is missing.

Q: Can I customize fonts and colors?
A: Yes! After installation, edit ~/.config/ghostty/config
   See INSTALLATION-GUIDE.md (Advanced Configuration section).

Q: Which font should I choose?
A: Noto Sans Mono CJK KR is recommended for terminals.
   The script offers 5 options during setup.

Q: What font size should I use?
A: Default is 12pt. Try 10-11 for Retina displays,
   12-14 for regular displays, 16-18 for accessibility.

Q: Will this work with my shell?
A: Yes! Works with bash, zsh, and other shells.
   The setup is terminal-agnostic.

TROUBLESHOOTING QUICK REFERENCE
================================================================================

If fonts don't display:
  → Check: ls ~/Library/Fonts | grep -i noto
  → Refresh: fc-cache -fv
  → Restart: killall ghostty
  → Reconfigure: ./apply-ghostty-config.sh

If Ghostty not found:
  → Install: brew install ghostty
  → Or: https://github.com/ghostty-org/ghostty

If scripts not executable:
  → Fix: chmod +x *.sh

If configuration fails:
  → Check logs: cat *.log | grep ERROR
  → Run diagnostics: ./verify-setup.sh
  → See detailed help: INSTALLATION-GUIDE.md (Troubleshooting section)

For more help:
  → QUICK-START.md for quick answers
