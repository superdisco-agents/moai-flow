# MoAI-Flow Agent Catalog

> Complete list of 54 agents organized by category

## Overview

MoAI-Flow provides 54 specialized agents organized into distinct categories. This contrasts with MoAI's 56 agents organized in a 5-tier hierarchy.

## Unique to MoAI-Flow

These agent types don't exist in MoAI:

1. **Swarm Coordination** (5 agents)
2. **Consensus & Distributed** (7 agents)
3. **Multi-repo Management** (`multi-repo-swarm`)
4. **Code Review Swarm** (distributed review)
5. **Neural/Adaptive** (`smart-agent`)
